# U1A4_AlgoritmosOrdenamiento
Comparación de rendimiento entre Bubble Sort y Quicksort en Python

## Descripción

Este proyecto implementa y evalúa el rendimiento de los algoritmos de ordenamiento Bubble Sort y Quicksort utilizando Python. Se realizaron pruebas controladas en VS Code para comparar su eficiencia en distintos tamaños de datos (100, 1000 y 10000) y escenarios 
(Lista aleatoria y lista invertida).

## Objetivo

Comparar experimentalmente el rendimiento de Bubble Sort y Quicksort mediante mediciones de tiempo de ejecución, analizando su comportamiento y eficiencia.

## Requisitos

- Python 3.10 o superior, en este caso, 3.13
- Visual Studio Code (opcional)
